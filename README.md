# 🚀 LeetCode Metrics Dashboard

A modern and interactive dashboard to visualize LeetCode statistics.  
This project helps developers track their coding progress using charts and analytics.

## 📌 Overview

LeetCode Metrics Dashboard allows users to enter their LeetCode username and view their coding performance in a clean and visually appealing dashboard. It provides insights such as problem-solving statistics, difficulty distribution, and submission analysis.

The goal of this project is to make developer progress tracking more engaging and data-driven.

---

## ✨ Features

- 🔍 Search LeetCode profile by username
- 📊 Difficulty-wise problem distribution (Easy / Medium / Hard)
- 📈 Visual charts for problem-solving progress
- 📉 Acceptance rate insights
- 📋 Total solved problems and submission statistics
- 🎨 Clean and responsive dashboard UI
- ⚡ Fast data fetching using API integration

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Dipakhyalij/LeetCodeMetrixProject.git
```

2. Navigate to the project folder

```bash
cd leetcode-metrics-dashboard
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

5. Open in browser

```
http://localhost:5173
```

---

## 📊 How It Works

1. User enters a LeetCode username.
2. The application fetches user statistics from the API.
3. Data is processed and displayed in charts and analytics cards.
4. Users can easily visualize their coding progress.
